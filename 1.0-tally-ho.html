Tally Ho!

To tally a box means to systematically count the items (or points) inside.



The process seems obvious:
  1. assign an initial tally value of 0
  2. unpack each point (systematically)
    1. assign the current tally value to the current point
    2. increase the tally by 1
  3. collect the final tally
  4. assert that the final tally is equal to the volume of the box
At the end of the process, each point will have been assigned a unique tally value, from 0 up to one less than the volume of the box - each point is identified by a unique index value in the range.

This article will consider:
  • different ways to tally/index a box using place value tables
  • resonances between place value tables captured in permutations
  • group actions that manipulate resonances
  • a physical interpretation of resonances as energy quanta
