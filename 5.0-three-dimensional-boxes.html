Three dimensional boxes

There are 6 ways to unpack the three-dimensional box 7 x 5 x 3 corresponding with 6 permutations of the 3 bases.

Selecting one permutation and then rotating the bases provides another two; these permutations are labelled "up" using an up arrow 🠅.

The reversal of each of these then provides the remaining three, and these permutations are labelled "down" using an arrow 🠇.

The indexes produced by each way of unpacking (place values table) can be compared with every other one (including themselves) to create 36 permutations of 105 points (the box volume).
